# CursorSync — Отслеживание курсора

Модуль для Topomatic Robur. Синхронизация курсора между видом плана и продольным профилем трассы.

## Возможности

- Автоматическая синхронизация позиции курсора между планом и продольным профилем
- Поддержка одновременной работы с несколькими видами

## Совместимость

| Продукт | Поддержка |
|---------|-----------|
| Топоматик Robur — Автомобильные дороги | ✓ |
| Топоматик Robur — Железные дороги | ✓ |

**Минимальная версия Robur:** 16.0.62.12

## Установка

Через менеджер модулей **ABR \| Библиотека модулей** (требуется модуль [abr-modules](https://github.com/Y-Abramov/AbrModules)):

1. Открыть **ABR → Модули → Управление модулями**
2. Найти **CursorSync** в каталоге
3. Нажать **Установить** и перезапустить Robur
